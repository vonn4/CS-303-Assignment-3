# CS-303-Assignment-3
After starting the program, follow these steps to test each functionality:

Input List:

First, enter up to 10 integers when prompted. This list will be used in each test.
To enter fewer than 10 integers, type -1 to end input early.
Main Menu:

The main menu will then appear, allowing you to choose a functionality:
1 for Queue Operations
2 for Recursive Search
3 for Recursive Insertion Sort
4 to Exit the program
Return to Main Menu or Exit:

After each test, the program will prompt you to return to the main menu or exit.
